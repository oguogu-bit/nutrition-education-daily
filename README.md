# 栄養教育デイリーノート

Claude Code によって毎日自動生成される栄養教育マテリアルです。

## 内容

毎朝 7:00 に以下のコンテンツが自動生成・pushされます：

- **今日の栄養素**：働きと効果、推奨摂取量（日本食事摂取基準2020）
- **豊富な食材 TOP5**：100gあたりの含有量
- **調理・保存のコツ**：栄養損失を最小化する方法
- **吸収効率UP の食べ合わせ**：科学的根拠に基づく組み合わせ
- **吸収を妨げる食べ合わせ**：注意すべき組み合わせ
- **今日のレシピ**：栄養保持を意識した調理アレンジ付き
