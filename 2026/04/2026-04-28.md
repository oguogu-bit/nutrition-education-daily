

# 疲労回復と栄養 — 栄養教育ノート 2026-04-28

## 今日の栄養素：ビタミンB群・クエン酸回路・抗酸化物質と疲労回復の科学

> 📅 日付の日（28日）→ **よくある誤解・吸収効率・組み合わせの最適化**にフォーカス

### 働きと効果

疲労とは、エネルギー産生の低下・酸化ストレスの蓄積・神経伝達の乱れが複合的に絡む生体反応です。エネルギー通貨であるATPは、クエン酸回路（TCA回路）と電子伝達系で産生されますが、この代謝には**ビタミンB1（糖質→アセチルCoA変換）、B2（FADとしてコハク酸脱水素酵素の補因子）、ナイアシン（NADとして複数段階の酸化還元）**が不可欠です。これらが不足するとピルビン酸が蓄積し、嫌気的代謝へ傾斜して疲労感が増大します。さらに、ATP産生時に生じる活性酸素（ROS）はビタミンCやポリフェノールなどの抗酸化物質で中和されます。よくある誤解として「栄養ドリンクのカフェインで疲労が回復する」というものがありますが、カフェインはアデノシン受容体をブロックして疲労感を"マスク"するだけであり、根本的なエネルギー代謝の改善にはB群の充足と十分な睡眠が必須です。

### 調理・保存による栄養素の変化

ビタミンB1は**水溶性かつ熱に弱い**二重のハンデを抱えています。

- **茹でる**：損失率 30〜50%。茹で汁に溶出するため、スープごと摂取が理想
- **炒める**：損失率 10〜20%。短時間高温が最適。豚肉の生姜焼きはB1保持に優れた調理法
- **煮る**：損失率 40〜60%。ただし煮汁ごと食べるカレーやシチューなら実質損失は少ない
- **電子レンジ**：損失率 10〜15%。水を使わないため最も保持率が高い

**よくある誤解：「にんにくと一緒に食べればB1は壊れない」**
→ 正確には、にんにくの**アリシン**がB1と結合して**アリチアミン**に変化し、脂溶性となることで腸管からの吸収率が飛躍的に向上（通常の約5〜6倍）します。B1が「壊れない」のではなく「吸収されやすい形に変換される」が正しい理解です。

ビタミンCについては、小松菜を茹でた場合の損失率は約40〜50%ですが、**短時間の塩茹で（1分以内）**で損失を20〜30%に抑えることが可能です。

### 吸収効率を上げる食べ合わせ

1. **ビタミンB1 ＋ アリシン（にんにく・玉ねぎ・ニラ）**
   アリシンがB1と結合してアリチアミンとなり、脂溶性に変化。腸管吸収率が約5〜6倍に向上し、体内での持続時間も延長されます。豚肉＋にんにくの組み合わせは科学的に裏付けられた「疲労回復の王道」です。

2. **クエン酸 ＋ ミネラル（鉄・カルシウム・マグネシウム）**
   クエン酸はミネラルをキレート（包み込み）して可溶性を高め、小腸での吸収効率を向上させます。梅干し＋小松菜、レモン汁＋ほうれん草のように酸味ある食材との組み合わせが有効です。

3. **ビタミンC ＋ 鉄分（非ヘム鉄）**
   ビタミンCが三価鉄（Fe³⁺）を二価鉄（Fe²⁺）に還元し、吸収率を約3〜6倍に向上させます。疲労時は鉄不足による酸素運搬能低下も一因となるため、鉄＋Cの同時摂取は疲労の根本改善に寄与します。

### 吸収を妨げる食べ合わせ（注意）

1. **タンニン（緑茶・紅茶・コーヒー）＋ 鉄分**
   タンニンが非ヘム鉄と不溶性複合体を形成し、吸収率を最大60%低下させます。食事中・食後30分以内のお茶・コーヒーは避け、食間に飲むのが理想です。

2. **アルコール ＋ ビタミンB1**
   アルコール代謝にB1が大量消費されるため、飲酒習慣のある人は慢性的なB1不足に陥りやすく、疲労が抜けにくくなります。いわゆる「飲んだ翌日のだるさ」の一因はB1枯渇です。

3. **過剰な糖質 ＋ ビタミンB1**
   糖質の代謝（ピルビン酸→アセチルCoA）にB1が必須のため、甘いジュースや菓子パンを大量に摂ると相対的なB1不足が生じ、かえって疲労感が増します。「甘いもので疲れを取る」は逆効果になりうる典型的な誤解です。

## 今日のレシピ：小松菜のナムル

### 栄養情報（1人前）

|--------|---|

| タンパク質 | 3 g |
| 脂質 | 4 g |

| カルシウム | 140 mg |
| ビタミンC | 39 mg |
| ビタミンD | 0 μg |
| ビタミンB1 | 0.07 mg |
| オメガ3 | 0.1 g |

### 材料（1人前）

- 小松菜 150g
- ごま油 小さじ1.5
- 塩 小さじ0.3
- にんにく少々
- 白ごま 小さじ1

### 作り方

1. 小松菜を塩茹でして冷水で冷やす
2. 水気を絞り3cm幅に切る
3. ごま油・塩・にんにく・ごまで和える

### このレシピの栄養学的ポイント

小松菜はカルシウム（140 mg）と鉄分（1.8 mg）を同時に摂取できる優れた緑黄色野菜です。今日のテーマ「疲労回復」の観点では、**ビタミンC（39 mg）が非ヘム鉄の吸収を促進**し、酸素運搬能の向上を通じて疲労の根本原因にアプローチします。さらに、にんにくに含まれるアリシンは微量ながらもB1の吸収効率を高める作用があり、作り置き（冷蔵3日）可能なため、疲労時にも手軽に栄養を補えます。

### 栄養保持・吸収効率UPのためのアレンジ提案

1. **レモン汁ちょい足しナムル**: 仕上げにレモン汁小さじ1を加えることで、ビタミンCの追加供給（約5 mg）と同時にクエン酸が鉄のキレート吸収を促進。非ヘム鉄の吸収率が2〜3倍向上し、疲労時の酸素運搬能改善に貢献します。

2. **豚しゃぶ小松菜ナムル**: 茹でた豚もも薄切り肉（50g）を加えることで、ビタミンB1を約0.48 mg追加。小松菜のにんにくとB1の「アリチアミン効果」で吸収効率がさらに向上し、クエン酸回路の稼働を本格的にサポートする疲労回復副菜に格上げできます。

3. **桜えびトッピング**: 乾燥桜えび大さじ1（5g）を散らすことで、カルシウムが約100 mg追加されるだけでなく、動物性のヘム鉄も微量追加。さらに桜えびの旨味が塩分を控える助けとなり、ナトリウム過剰（現在300 mg）の軽減にもつながります。
