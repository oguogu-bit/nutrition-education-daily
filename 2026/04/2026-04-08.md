

# ビタミンD — 栄養教育ノート 2026-04-08

## 今日の栄養素：ビタミンD

### 働きと効果

ビタミンDは脂溶性ビタミンの一つで、体内では**ホルモン様の働き**を持つ特殊な栄養素です。最も重要な役割は**小腸でのカルシウム・リンの吸収促進**で、ビタミンDが不足するとカルシウムをいくら摂取しても吸収効率が著しく低下します。さらに近年の研究では、**免疫調節機能**が注目されており、自然免疫・獲得免疫の両方に関与し、感染症リスクの低減や自己免疫疾患との関連が報告されています。欠乏すると、成人では**骨軟化症**、小児では**くる病**を引き起こし、骨痛・筋力低下・易骨折性が出現します。また、冬季うつ（季節性感情障害）との関連も指摘されています。一方、脂溶性であるため体内に蓄積しやすく、過剰摂取では**高カルシウム血症**を招き、腎臓障害・血管石灰化・食欲不振・嘔吐などの症状が出ることがあります。サプリメントによる過剰摂取には特に注意が必要です。

### 調理・保存による栄養素の変化

ビタミンDは脂溶性ビタミンの中でも**比較的熱に安定**しており、通常の加熱調理（焼く・煮る・蒸す）での損失率は概ね**10〜20%程度**と小さいのが特徴です。

- **焼き調理**：損失率は約10%。魚の塩焼きなどは効率よくビタミンDを摂取できる調理法です
- **煮る**：ビタミンDは水に溶けにくいため煮汁への溶出は少ないですが、脂と共に多少流出します。煮汁ごと食べる料理（味噌汁・シチュー）が理想的です
- **揚げる**：高温の油で揚げると15〜20%程度の損失がありますが、脂溶性のため油との相性は良好です
- **干ししいたけの特殊性**：乾燥前のしいたけに含まれるエルゴステロール（プロビタミンD₂）は、**紫外線（UV-B）照射**によりビタミンD₂に変換されます。調理前に**かさの裏面を上にして30分〜1時間日光に当てる**と、含有量が数倍に増加します
- **保存**：遮光・密閉保存が基本。光（特に紫外線）による分解があるため、魚の干物なども直射日光を避けて保存しましょう

### 吸収効率を上げる食べ合わせ

1. **ビタミンD ＋ 脂質 → 吸収率が大幅向上**
   ビタミンDは脂溶性のため、脂質と一緒に摂取すると胆汁酸によるミセル形成が促進され、小腸での吸収効率が**2〜3倍**に向上します。鮭のオリーブオイルソテーや、しらすにごま油をかけるなど、良質な脂質との組み合わせが効果的です。

2. **ビタミンD ＋ カルシウム ＋ ビタミンK2 → 骨への沈着が最適化**
   ビタミンDがカルシウムの吸収を促進し、ビタミンK2がカルシウムを骨に運ぶオステオカルシンを活性化します。**鮭＋チーズ＋納豆**の組み合わせはこの3栄養素を同時に摂取できる理想的な食べ合わせです。

3. **ビタミンD ＋ マグネシウム → ビタミンDの活性化を促進**
   ビタミンDは肝臓と腎臓で活性型に変換されますが、この過程にマグネシウムが必要です。マグネシウムが不足するとビタミンDの代謝効率が低下するため、**魚料理＋ほうれん草やアーモンド**などマグネシウム豊富な食材との組み合わせが推奨されます。

### 吸収を妨げる食べ合わせ（注意）

1. **食物繊維の過剰摂取との同時摂取**：大量の食物繊維（特に不溶性食物繊維）は、脂溶性ビタミンを吸着して便と共に排泄してしまう可能性があります。サプリメントでビタミンDを摂る場合、食物繊維の多い食事と時間をずらすと安心です。

2. **オルリスタット（脂肪吸収抑制薬）**：脂質の吸収を阻害する薬剤は、脂溶性ビタミン全般の吸収も低下させます。該当する薬を服用中の方は、医師に相談しましょう。

3. **ミネラルオイル（流動パラフィン）**：便秘薬として使われることがありますが、脂溶性ビタミンを溶かし込み、吸収されないまま排泄されてしまいます。常用している方は注意が必要です。

## 今日のレシピ：豚ヒレカツ（オーブン焼き）

### 栄養情報（1人前）

| 栄養素 | 量 |
|--------|-----|
| カロリー | 340 kcal |
| タンパク質 | 28 g |
| 脂質 | 14 g |
| 炭水化物 | 20 g |
| 食物繊維 | 0.5 g |
| 鉄分 | 0.9 mg |
| カルシウム | 20 mg |
| ビタミンC | 0 mg |
| ビタミンD | 0.1 μg |
| ビタミンB1 | 0.78 mg |
| オメガ3 | 0.1 g |

### 材料（1人前）

- 豚ヒレ肉 150g
- パン粉 大さじ3
- 卵 1/2個
- 薄力粉 大さじ1
- 塩こしょう 少々
- オリーブ油 大さじ1

### 作り方

1. 豚ヒレを叩いて薄くし塩こしょう
2. 薄力粉→卵→パン粉の順にまぶす
3. オリーブ油をかけ200℃のオーブンで20分

### このレシピの栄養学的ポイント

このレシピは揚げずにオーブンで焼くことでカロリーを抑えつつ、豚ヒレ肉由来の豊富な**ビタミンB1（0.78mg）**を効率よく摂取できます。ただし、今日のテーマである**ビタミンDは0.1μgとごく微量**です。豚肉はビタミンD供給源としてはほぼ期待できないため、副菜や汁物でビタミンDを補う工夫が重要です。オリーブ油の脂質が脂溶性ビタミンの吸収を助けるため、副菜にビタミンDを含む食材を加えれば、吸収効率を高めることができます。

### 栄養保持・吸収効率UPのためのアレンジ提案

1. **しらすチーズトッピング**：焼き上がりの最後3分で、しらす干し大さじ2＋とろけるチーズ20gを載せてオーブンで溶かします。しらすのビタミンD（約12μg追加）とチーズのカルシウムにより、**ビタミンD＋カルシウムの黄金コンビ**が実現。さらにオリーブ油の脂質がビタミンDの吸収を促進します。

2. **干ししいたけ入りパン粉衣**：干ししいたけを粉末にしてパン粉に混ぜ込みます（干ししいたけ2枚分）。調理前にしいたけを日光に30分当てておくと、ビタミンD₂含有量がさらにアップ。衣に旨味も加わり、**減塩効果**も期待できます。

3. **きのこクリームソースを添える**：まいたけ・エリンギをバターでソテーし、牛乳100mlで軽くソースにしてヒレカツに添えます。まいたけはビタミンD含有量がきのこ類の中でもトップクラス（約4.9μg/100g）。牛乳のカルシウムとの相乗効果で、**骨の健康をトータルサポート**する一皿に変わります。

## 今日のまとめ

- **ビタミンDは「日光＋食事」の二刀流**で確保する栄養素。4月は紫外線量が増える時期なので、1日15〜30分の適度な日光浴（手の甲・前腕程度の露出で十分）を意識しましょう
- **脂質と一緒に摂る**ことで吸収効率が大幅アップ。魚やきのこは油を使った調理法（ソテー・オーブン焼き）がおすすめです
- 豚肉はビタミンDが少ないため、**副菜にしらす・きのこ・卵**を組み合わせて「ビタミンDの死角」を補いましょう
