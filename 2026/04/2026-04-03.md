

# 炭水化物 — 栄養教育ノート 2026-04-03

## 今日の栄養素：炭水化物（糖質＋食物繊維）

### 働きと効果

炭水化物は体内でブドウ糖に分解され、**脳と筋肉の最も重要なエネルギー源**として機能します。特に脳は通常時、エネルギーの約20%を消費し、そのほぼ全てをブドウ糖に依存しています。炭水化物が不足すると、集中力低下・疲労感・低血糖症状（めまい・冷や汗）が現れ、体はタンパク質を分解してエネルギーを補おうとするため**筋肉量の減少**にもつながります。一方、過剰摂取は血糖値の急上昇→インスリン大量分泌→体脂肪蓄積という経路で**肥満・2型糖尿病・脂質異常症**のリスクを高めます。近年注目されるのは「糖質の量」だけでなく「糖質の質」——すなわちGI値（グリセミック・インデックス）や食物繊維との同時摂取による**血糖応答の違い**が健康に大きく影響するという点です。

### 調理・保存による栄養素の変化

炭水化物そのものは加熱で「消失」しませんが、**形態が変化**することで消化吸収速度が大きく変わります。

- **加熱（糊化）**：生のデンプンは加熱すると糊化（α化）し、消化酵素が作用しやすくなります。炊飯した米が消化しやすいのはこのためです。
- **冷却（老化）**：糊化したデンプンを冷ますと一部が**レジスタントスターチ（難消化性デンプン）**に変化します。冷やご飯・冷製パスタは食物繊維に似た働きをし、血糖値上昇を約20〜30%抑える研究結果があります。
- **長時間加熱・高温調理**：パンのトーストや揚げ物の衣など、高温で長時間加熱すると**アクリルアミド**（発がん性が指摘される物質）が生成されるリスクがあります。焦がしすぎに注意。
- **保存**：冷凍ご飯は再加熱後もレジスタントスターチが白米より多く残る傾向があり、血糖管理の観点からは**まとめ炊き→冷凍→再加熱**も有効な手法です。

### 吸収効率を上げる食べ合わせ

> ※ 炭水化物の場合、「吸収を上げる」よりも**「血糖値の急上昇を防ぎながら効率よくエネルギーに変える」**組み合わせが重要です。

1. **炭水化物 ＋ ビタミンB1（豚肉・大豆）**
   糖質をエネルギーに変換するTCA回路の入口で、ビタミンB1は**補酵素（チアミンピロリン酸）**として不可欠です。B1が不足すると糖質を摂っても効率的にエネルギー化できず、疲労感や乳酸の蓄積を招きます。ご飯＋豚の生姜焼きは理にかなった組み合わせです。

2. **炭水化物 ＋ 食物繊維（野菜・海藻）**
   食物繊維が胃での食物の滞留時間を延ばし、小腸での糖質吸収速度を緩やかにします。**食前にサラダや味噌汁（わかめ等）**を食べるだけで、食後血糖値のピークを約30〜40%抑制できるとの報告があります。

3. **炭水化物 ＋ 酢（酢酸）**
   酢に含まれる酢酸は、胃排出速度を遅らせ、筋肉でのグリコーゲン再合成を促進します。食事に**酢の物や南蛮漬け**を添えることで、食後血糖値の上昇が約20〜35%抑制されるとする研究があります。

### 吸収を妨げる食べ合わせ（注意）

1. **炭水化物の単独・大量摂取（おにぎりだけ、菓子パンだけ）**
   タンパク質・脂質・食物繊維を伴わない糖質の単独摂取は、血糖値の急上昇（血糖スパイク）を引き起こします。インスリンが過剰分泌され、その後の**反応性低血糖**で眠気・集中力低下を招きます。

2. **糖質 ＋ 大量の精製脂質（ドーナツ・ポテトチップス等）**
   精製糖質と飽和脂肪酸の同時大量摂取は、インスリン抵抗性を悪化させ、中性脂肪の上昇を促進します。これは**最も肥満リスクが高い組み合わせ**のひとつです。

3. **過度なアルコール ＋ 糖質**
   アルコールはビタミンB1の吸収を阻害し、消耗を促進します。飲酒時に締めのラーメンなど糖質を大量に摂ると、B1不足により糖質代謝が滞り、エネルギー変換効率が著しく低下します。

## 今日のレシピ：サバの味噌煮

### 栄養情報（1人前）

| 栄養素 | 量 |
|--------|---|
| カロリー | 300 kcal |
| タンパク質 | 25 g |
| 脂質 | 16 g |
| 炭水化物 | 12 g |
| 食物繊維 | 0.5 g |
| 鉄分 | 1.2 mg |
| カルシウム | 18 mg |
| ビタミンC | 0 mg |
| ビタミンD | 5.5 μg |
| ビタミンB1 | 0.22 mg |
| オメガ3 | 2.8 g |

### 材料（1人前）

- サバ 1切れ（120g）
- 味噌 大さじ1.5
- みりん 大さじ1
- 醤油 小さじ1
- 砂糖 小さじ1
- 生姜 1片
- 水 150ml

### 作り方

1. サバに熱湯をかけ臭みを取る
2. 調味料と水を合わせ煮立てる
3. サバを入れ落し蓋をして15分煮る
4. 煮汁を煮詰めて完成

### このレシピの栄養学的ポイント

サバはEPA/DHAなどのオメガ3脂肪酸が**2.8g**と極めて豊富で、慢性炎症の抑制・中性脂肪の低下・脳機能の維持に寄与します。さらにビタミンDが5.5μgと、1日の目安量（8.5μg）の約65%をカバー。今日のテーマである炭水化物との関連では、本レシピの炭水化物は調味料由来の**12g**と控えめで、**主食（ご飯）と組み合わせることで糖質＋ビタミンB1（0.22mg含有）の相乗効果**が期待できます。サバのB1が白米の糖質代謝を助け、効率的なエネルギー変換を支えてくれる一品です。

### 栄養保持・吸収効率UPのためのアレンジ提案

1. **玄米ご飯との組み合わせ**：白米を玄米に置き換えることで、食物繊維が約6倍に増加し、炭水化物の吸収速度が緩やかになります。玄米自体にもビタミンB1が白米の約5倍含まれるため、糖質→エネルギー変換の効率がさらに向上します。

2. **根菜の付け合わせ追加**：大根おろしやごぼうの煮物を添えることで、食物繊維をプラスし血糖値の急上昇を防ぎます。特に大根おろしは消化酵素ジアスターゼを含み、炭水化物の消化を助けるとともに、サバの脂質の消化もサポートします。

3. **食べる順番の工夫（味噌汁→副菜→サバ→ご飯）**：味噌煮の前にわかめの味噌汁や酢の物を先に食べることで、水溶性食物繊維や酢酸が胃腸での糖質吸収を緩やかにします。同じ食事内容でも**食べる順番を変えるだけで食後血糖値のピークを約30%抑制**できるという研究報告があります。

## 今日のまとめ

- **炭水化物は「量」より「質」と「食べ方」が重要** — GI値の低い食品を選び、食物繊維やタンパク質と一緒に摂ることで血糖スパイクを防げる
- **ビタミンB1は糖質代謝の鍵** — サバや豚肉などB1を含むおかずとご飯を組み合わせることで、糖質が効率よくエネルギーに変わる
- **冷やご飯のレジスタントスターチを活用** — ご飯を一度冷ますだけで食物繊維に似た効果が生まれ、血糖管理に役立つ
