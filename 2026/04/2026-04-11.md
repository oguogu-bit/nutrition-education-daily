

# ビタミンA/βカロテン — 栄養教育ノート 2026-04-11

## 今日の栄養素：ビタミンA（レチノール）/ βカロテン（プロビタミンA）

### 働きと効果

ビタミンAは、視覚機能の維持（網膜のロドプシン合成）、皮膚・粘膜の健康維持、免疫機能の調節、細胞の分化・増殖に不可欠な脂溶性ビタミンです。特に暗所での視力（暗順応）に深く関わり、欠乏すると**夜盲症**や**皮膚の乾燥・角化**、**感染症への抵抗力低下**が生じます。一方、動物性食品由来のレチノール（既成ビタミンA）は体内に蓄積されやすく、過剰摂取により**頭痛・吐き気・肝障害**、妊娠初期では**胎児の奇形リスク**が上昇します。植物性食品に含まれるβカロテンは、体内で必要量だけビタミンAに変換される「プロビタミンA」であり、過剰症のリスクが極めて低い安全な供給源です。βカロテン自体にも強力な**抗酸化作用**があり、活性酸素の消去を通じて老化防止や生活習慣病予防に寄与します。

### 調理・保存による栄養素の変化

βカロテンは**熱に比較的安定**な栄養素であり、通常の加熱調理（炒め物・煮物・蒸し物）による損失は10〜20％程度と少ないのが特長です。むしろ加熱により**細胞壁が破壊**されることで、生食時よりもβカロテンの体内利用率が大幅に向上します。

- **生食**：吸収率は約8〜10％と低い（細胞壁にβカロテンが閉じ込められるため）
- **油で炒める**：吸収率が約60〜70％まで上昇（脂溶性のため油との親和性が高い）
- **煮る**：煮汁に一部溶出するが、煮汁ごと摂取すれば問題なし
- **切り方**：細かく刻む・すりおろすほど細胞壁が壊れ、吸収率が向上する
- **保存**：光と酸素に弱いため、にんじんなどは新聞紙に包み冷暗所で立てて保存。冷凍保存ではβカロテンの損失はほぼない

### 吸収効率を上げる食べ合わせ

1. **βカロテン ＋ 油脂（オリーブオイル・ごま油など）**
   βカロテンは脂溶性のため、少量の油脂と一緒に摂ると吸収率が**生食の6〜7倍**に跳ね上がります。にんじんしりしりやかぼちゃのバターソテーは理にかなった調理法です。

2. **βカロテン ＋ ビタミンE（アーモンド・アボカドなど）**
   ビタミンEはβカロテンの酸化を防ぎ、体内での抗酸化力を相乗的に高めます。にんじんとアーモンドのサラダなどが好例です。

3. **βカロテン ＋ 亜鉛（牡蠣・牛肉など）**
   ビタミンAが肝臓から血中に放出される際に**レチノール結合タンパク質（RBP）**が必要で、その合成には亜鉛が不可欠です。亜鉛不足はビタミンAの体内利用効率を低下させます。

### 吸収を妨げる食べ合わせ（注意）

1. **ミネラルオイル（鉱物油）との同時摂取**
   下剤などに含まれるミネラルオイルは脂溶性ビタミンの吸収を著しく阻害します。ビタミンA豊富な食事の前後は使用を避けましょう。

2. **極端な低脂肪食との組み合わせ**
   脂質をほとんど含まない食事では、βカロテンの吸収率が極端に低下します。サラダにはドレッシング（油脂含有）を適量かけることが重要です。「ノンオイルドレッシング＋にんじんサラダ」はβカロテンの観点からは非効率的です。

3. **食物繊維の過剰摂取との同時摂取**
   大量の不溶性食物繊維が消化管内でβカロテンを吸着し、排泄を早めてしまう可能性があります。通常量であれば問題ありませんが、サプリメント等で食物繊維を大量摂取する場合は時間をずらしましょう。

## 今日のレシピ：切り干し大根の煮物

### 栄養情報（1人前）

| 栄養素 | 量 |
|--------|---|
| カロリー | 70 kcal |
| タンパク質 | 2 g |
| 脂質 | 1 g |
| 炭水化物 | 13 g |
| 食物繊維 | 3.5 g |
| 鉄分 | 1.1 mg |
| カルシウム | 100 mg |
| ビタミンC | 0 mg |
| ビタミンD | 0 μg |
| ビタミンB1 | 0.04 mg |
| オメガ3 | 0 g |

### 材料（1人前）

- 切り干し大根 30g
- 人参 1/3本
- 油揚げ 1/4枚
- 出汁 200ml
- 醤油 大さじ1.5
- みりん 大さじ1
- 砂糖 小さじ1

### 作り方

1. 切り干し大根を水で戻す
2. 人参・油揚げを細切りにする
3. 出汁で煮て調味料を加え15分煮る

### このレシピの栄養学的ポイント

本レシピには**人参**が含まれており、これが今日のテーマであるβカロテンの供給源となります。人参1/3本（約50g）には約4,300μgのβカロテンが含まれ、レチノール活性当量で約360μgRAEに相当します。また、切り干し大根はカルシウム（100mg）と食物繊維（3.5g）が豊富で、**常備菜として冷蔵5日保存可能**な実用性の高い一品です。油揚げに含まれる油脂が、人参のβカロテン吸収を助ける役割も果たしています。

### 栄養保持・吸収効率UPのためのアレンジ提案

1. **ごま油で炒めてから煮る（βカロテン吸収UP）**: 具材を先にごま油小さじ1で軽く炒めてから出汁を加えて煮ることで、人参のβカロテン吸収率が大幅に向上します。油揚げだけでは油脂量が十分でないため、この一手間で脂溶性ビタミンの利用効率が格段に上がります。

2. **ほうれん草・小松菜を追加（βカロテン＋カルシウム増強）**: 仕上げの直前にほうれん草や小松菜をさっと加えることで、βカロテンとカルシウムの両方を底上げできます。加熱時間を短くすることで色味と栄養素を保持しつつ、煮汁に溶け出した栄養も一緒に摂れます。

3. **すりごまを仕上げにかける（ビタミンE＋抗酸化力UP）**: すりごま大さじ1を盛り付け時にかけることで、ビタミンEの抗酸化作用がβカロテンの酸化分解を防ぎ、体内での抗酸化効果を相乗的に高めます。さらに、ごまに含まれるセサミンも抗酸化物質として働き、風味も豊かになります。

## 今日のまとめ

- **βカロテンは「油と一緒に」が鉄則** — 生食では吸収率わずか8〜10％だが、油脂と一緒に調理・摂取すると60〜70％まで向上する。煮物でも炒め工程を加えるだけで効果大
- **動物性レチノールとβカロテンを使い分ける** — レバーなどの動物性食品は高効率だが過剰摂取リスクあり。日常的にはにんじん・かぼちゃなど植物性βカロテンで安全に補給し、レバーは月1〜2回の頻度に
- **常備菜の切り干し大根煮は「ごま油炒め＋すりごまトッピング」でβカロテン吸収を最大化** — 今日のレシピにひと工夫加えるだけで、人参由来のβカロテンの体内利用率を飛躍的に改善できる
