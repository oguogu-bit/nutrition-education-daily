

# ビタミンC — 栄養教育ノート 2026-04-07

## 今日の栄養素：ビタミンC（アスコルビン酸）

### 働きと効果

ビタミンCは強力な**抗酸化作用**を持つ水溶性ビタミンで、体内で発生する活性酸素を消去し、細胞の酸化ダメージを防ぎます。最も重要な機能の一つが**コラーゲン合成**への関与です。コラーゲンは皮膚・血管壁・骨・腱などの結合組織の主要タンパク質であり、ビタミンCはプロリンやリジンの水酸化反応に必須の補因子として働きます。欠乏すると歯茎の出血、皮下出血、傷の治りの遅延が起こり、重症化すると**壊血病**に至ります。また、**非ヘム鉄の吸収を2〜3倍に促進**し、免疫細胞（好中球・リンパ球）の機能を活性化させることで感染症への抵抗力を高めます。水溶性のため体内に蓄積されにくく、過剰摂取分は尿中に排泄されますが、一度に数グラム以上を摂ると下痢や腹痛、腎結石のリスクが報告されています。

### 調理・保存による栄養素の変化

ビタミンCは**熱・水・空気（酸素）・光**のすべてに弱い、最もデリケートなビタミンの一つです。

- **茹で調理**：水溶性のため茹で汁に溶出し、**40〜60%が損失**します。ブロッコリーを5分茹でると約50%減少。スープにして汁ごと摂れば損失を最小限にできます
- **炒め調理**：短時間の強火炒めなら**損失は15〜25%程度**に抑えられます。油でコーティングされることで酸化も防がれます
- **蒸し調理**：水に触れないため**損失は10〜20%程度**。最も栄養保持率の高い加熱法です
- **電子レンジ加熱**：短時間・少水で加熱できるため**損失は15〜25%程度**と比較的優秀
- **切り方**：細かく切るほど断面から酸化・流出が進みます。**大きめに切り、食べる直前に調理**するのが鉄則
- **保存**：冷蔵保存でも日数経過とともに減少。ほうれん草は室温で2日置くと約80%が失われます。**購入後は早めに消費するか、ブランチング後に冷凍保存**が有効です

### 吸収効率を上げる食べ合わせ

1. **ビタミンC ＋ 非ヘム鉄（植物性鉄分）**
   非ヘム鉄は三価鉄（Fe³⁺）の形で存在し、そのままでは吸収率がわずか2〜5%。ビタミンCが三価鉄を二価鉄（Fe²⁺）に還元することで、**吸収率が3〜6倍に向上**します。ほうれん草のおひたしにレモン汁をかける、小松菜の炒め物にパプリカを加えるなどが効果的です。

2. **ビタミンC ＋ ビタミンE**
   ビタミンEは脂溶性の抗酸化物質ですが、活性酸素を消去した後は「ビタミンEラジカル」となり機能を失います。ビタミンCがこのラジカルを還元し、**ビタミンEを再生**させます。アーモンド（ビタミンE）＋キウイ（ビタミンC）のサラダなどが理想的です。

3. **ビタミンC ＋ コラーゲン（タンパク質）**
   コラーゲン合成にはビタミンCが必須です。手羽先や豚足などコラーゲン豊富な食材を食べる際にレモンや野菜を添えることで、体内でのコラーゲン再合成が促進されます。

### 吸収を妨げる食べ合わせ（注意）

- **喫煙**：厳密には「食べ合わせ」ではありませんが、タバコ1本で約25 mgのビタミンCが消費されます。喫煙者は慢性的なビタミンC不足に陥りやすく、食事からの摂取では追いつかない場合があります
- **大量のアルコール**：アルコール代謝で発生するアセトアルデヒドの解毒にビタミンCが大量消費されます。飲酒時にはフルーツなどでビタミンCを補給するのが望ましいです
- **銅イオン（過剰な場合）**：銅の存在下でビタミンCは酸化が促進され、抗酸化力が低下します。通常の食事では問題になりませんが、サプリメントの併用時には注意が必要です

## 今日のレシピ：アジの塩焼き

### 栄養情報（1人前）

| 栄養素 | 量 |
|--------|---|
| カロリー | 180 kcal |
| タンパク質 | 22 g |
| 脂質 | 8 g |
| 炭水化物 | 0 g |
| 食物繊維 | 0 g |
| 鉄分 | 0.7 mg |
| カルシウム | 65 mg |
| ビタミンC | 0 mg |
| ビタミンD | 8 μg |
| ビタミンB1 | 0.12 mg |
| オメガ3 | 1.2 g |

### 材料（1人前）

- アジ 1尾
- 塩 小さじ1/2
- レモン 適量
- 大根おろし 適量

### 作り方

1. アジに塩をふり15分置く
2. グリルで両面焼く（各5分）
3. 大根おろしとレモンを添える

### このレシピの栄養学的ポイント

アジの塩焼きはビタミンDが8 μgと豊富で、1人前で成人の1日目安量（8.5 μg）のほぼ全量を摂取できます。EPA/DHA（オメガ3脂肪酸）も1.2 gと十分量含まれ、抗炎症作用や脳機能の維持に貢献します。**ただし、このレシピ単体ではビタミンCが0 mg**です。添えるレモンを意識的にしっかり搾ることで、ビタミンCの補給とともにアジに含まれる非ヘム鉄（0.7 mg）の吸収率も向上させることができます。

### 栄養保持・吸収効率UPのためのアレンジ提案

1. **レモンたっぷり搾り＋パセリ添え**：レモン1/4個（約10 mg のビタミンC）に加え、刻みパセリ（大さじ1で約5 mg）を大根おろしに混ぜることで、ビタミンC含有量が0→約15 mgに向上。鉄分の吸収促進にも直結します。パセリは生のまま仕上げに加えることでビタミンCの損失をゼロにできます。

2. **ピーマンとトマトの彩りマリネを副菜に**：赤ピーマン1/2個（約60 mgのビタミンC）＋ミニトマト3個（約15 mg）をオリーブオイルと酢でマリネにして添えれば、一食でビタミンCを約75 mg確保でき、推奨量の75%をカバー。非加熱なのでビタミンCの損失もありません。

3. **骨ごと食べられる小アジの南蛮漬けにアレンジ**：小ぶりのアジを使い、骨ごと素揚げ後に酢・玉ねぎ・ピーマンの南蛮酢に漬け込むことで、カルシウム摂取量が大幅に増加。酢に含まれる酢酸がカルシウムの溶出と吸収を促進し、ピーマンのビタミンCも加わります。玉ねぎは生のまま漬けるため、ビタミンCの損失も最小限です。

## 今日のまとめ

- **ビタミンCは「茹でない・切りすぎない・早く食べる」が三原則**。蒸し調理やレンジ加熱を活用し、水に溶け出す損失を防ぎましょう
- **魚料理にはレモンや生野菜を必ず添える習慣**をつけると、ビタミンCがほぼゼロの動物性食品でもビタミンCの恩恵（鉄吸収促進・抗酸化作用）を受けられます
- **ビタミンCはビタミンEを再生する「抗酸化リレー」のアンカー**。ナッツ類（E）＋果物（C）の組み合わせは最強の抗酸化コンビです
