

# 鉄欠乏性貧血の予防 — 栄養教育ノート 2026-04-29

## 今日の栄養素：鉄（よくある誤解・吸収効率・組み合わせの最適化）

### 働きと効果

鉄は赤血球中のヘモグロビンの構成成分として、全身の細胞に酸素を届ける中心的な役割を担います。体内の鉄は約3〜4gで、うち約60〜70%が「機能鉄」としてヘモグロビンやミオグロビンに、残りは「貯蔵鉄」としてフェリチンやヘモシデリンの形で肝臓・脾臓・骨髄に蓄えられています。鉄が不足すると、まず貯蔵鉄が減少し（潜在性鉄欠乏）、さらに進行するとヘモグロビン合成が低下して鉄欠乏性貧血を発症します。症状は倦怠感・動悸・息切れ・集中力低下・匙状爪（スプーンネイル）・氷食症など多岐にわたります。一方、鉄の過剰摂取は活性酸素の産生を促進し、肝障害や心筋障害を引き起こすリスクがあるため、サプリメントの安易な大量摂取は避けるべきです。日本人女性の約40%が潜在性鉄欠乏とも報告されており、日常の食事からの効率的な鉄摂取が極めて重要です。

### 調理・保存による栄養素の変化

鉄はミネラルであるため、ビタミンのように加熱で「壊れる」ことはありません。しかし、以下の調理条件で損失が生じます。

- **茹で調理**：葉物野菜の鉄分は茹で汁へ約20〜30%溶出します。蒸し調理やレンジ加熱にすると損失を5%以下に抑えられます。
- **鋳鉄鍋の活用**：鉄鍋や鉄フライパンで酸性の食材（トマトソース・酢を使う料理）を調理すると、鍋から溶出した鉄が食品に移行し、鉄摂取量を増やせることが研究で確認されています（最大で1食あたり+1〜3 mg）。
- **あく抜き・水さらし**：ほうれん草のシュウ酸除去のために茹でると鉄も流出するため、茹で時間は30秒〜1分を目安に最短にします。
- **保存中の変化**：乾物（切干大根・ドライプルーン）は水分が抜けることで100gあたりの鉄含有量が凝縮され、効率よく摂取できる形態です。

### 吸収効率を上げる食べ合わせ

**【よくある誤解を正す】** 「鉄分の多い食品をたくさん食べれば貧血は治る」は不正確です。非ヘム鉄の吸収率はわずか2〜5%であり、**吸収促進因子との組み合わせ**が決定的に重要です。

1. **非ヘム鉄 ＋ ビタミンC**：ビタミンCは非ヘム鉄（Fe³⁺）を還元型（Fe²⁺）に変換し、腸管での吸収率を最大6倍に高めます。小松菜のお浸し＋レモン汁、ほうれん草＋いちごのサラダなどが効果的です。食事中に50mg以上のビタミンCを摂ると有意な促進効果が確認されています。

2. **非ヘム鉄 ＋ MFP因子（肉・魚・鶏肉）**：動物性タンパク質に含まれるMFP因子（Meat, Fish, Poultry factor）は、非ヘム鉄の吸収を2〜3倍促進します。肉じゃが・魚介と野菜の炒め物など、動物性食品と植物性食品を同じ食事で摂ることが有効です。

3. **クエン酸・リンゴ酸などの有機酸**：酢・梅干し・柑橘類に含まれる有機酸はキレート作用により鉄をイオン化し、吸収を促進します。**今日のレシピ「紫キャベツの酢漬け」はまさにこの原理を活用した組み合わせです。**

### 吸収を妨げる食べ合わせ（注意）

**【よくある誤解を正す】** 「お茶を飲むと鉄がまったく吸収されなくなる」は過度な心配です。以下に正確な情報を示します。

- **タンニン（緑茶・紅茶・コーヒー）**：タンニンは非ヘム鉄と不溶性の錯体を形成し吸収を阻害しますが、その影響は「食事中の同時摂取」で最も顕著です。食事の前後30分〜1時間を避ければ影響は大幅に軽減されます。近年の研究では、十分なビタミンCを同時に摂ればタンニンの阻害効果は相殺されるとの報告もあります。
- **フィチン酸（全粒穀物・豆類の未加工品）**：強力なキレート剤として鉄と結合します。ただし、発酵（パン酵母・納豆）・浸水・発芽処理でフィチン酸は分解されるため、全粒粉パン（酵母発酵）や納豆は実は鉄の吸収に不利ではありません。
- **カルシウムの大量同時摂取**：カルシウムはヘム鉄・非ヘム鉄の両方の吸収を阻害します。鉄強化を意識した食事では、牛乳やカルシウムサプリメントは食間に摂るのが望ましいです。
- **食物繊維の大量摂取**：腸管内で鉄と結合し排泄を促進する可能性がありますが、通常の食事レベル（20〜25g/日）であれば問題ありません。

## 今日のレシピ：紫キャベツの酢漬け

### 栄養情報（1人前）

| 栄養素 | 量 |
|--------|---|
| カロリー | 40 kcal |
| タンパク質 | 1 g |
| 脂質 | 0 g |
| 炭水化物 | 8 g |
| 食物繊維 | 1.8 g |
| 鉄分 | 0.3 mg |
| カルシウム | 30 mg |
| ビタミンC | 55 mg |
| ビタミンD | 0 μg |
| ビタミンB1 | 0.04 mg |
| オメガ3 | 0 g |

### 材料（1人前）

- 紫キャベツ 150g
- 酢 大さじ2
- 砂糖 大さじ1
- 塩 小さじ0.5

### 作り方

1. 紫キャベツを千切りにして塩もみ
2. 水気を絞り酢・砂糖で和える
3. 1時間以上冷蔵庫で漬ける

### このレシピの栄養学的ポイント

このレシピ単体の鉄分は0.3mgと少量ですが、**真価は「副菜」としての鉄吸収促進力**にあります。1人前でビタミンCが55mgと非常に豊富で、これは鉄吸収促進に必要とされる50mgを超える量です。さらに酢に含まれる酢酸（有機酸）がキレート作用で非ヘム鉄の可溶化を助けます。つまり、鉄分を多く含む主菜（レバニラ炒め・あさりパスタ・牛肉の赤身など）と一緒にこの副菜を食べることで、食事全体の鉄吸収率を飛躍的に高める「吸収ブースター」として機能します。冷蔵5日保存可能で作り置きでき、毎食の鉄吸収効率を底上げできるコスパ最強の常備菜です。

### 栄養保持・吸収効率UPのためのアレンジ提案

1. **レモン汁追加アレンジ**: 酢の半量をレモン汁に置き換えることで、ビタミンCをさらに増強し、クエン酸による鉄キレート効果も加わります。ビタミンCは非ヘム鉄のFe³⁺→Fe²⁺還元に直接関与するため、鉄分豊富な主菜との組み合わせ時に最大の効果を発揮します。酸味もまろやかになり食べやすくなります。

2. **枝豆・蒸し大豆トッピング**: 蒸し大豆や枝豆を加えることで、1人前あたり鉄分+1.5mg、タンパク質+5g程度の強化が可能です。大豆の非ヘム鉄は紫キャベツのビタミンCと酢の有機酸によって吸収率が向上し、さらに植物性タンパク質も補完できる一石三鳥のアレンジです。蒸し加工によりフィチン酸も一部分解されています。

3. **ツナ缶（赤身）追加**: 油を切ったツナ缶（まぐろ赤身）を加えることで、ヘム鉄とMFP因子を同時にプラスできます。MFP因子は紫キャベツの非ヘム鉄の吸収も促進し、さらにツナ自体がヘム鉄（吸収率15〜35%）を含むため、この副菜単体の鉄補給力が大幅に向上します。タンパク質も加わり、バランスの取れた副菜に進化します。

## 今日のまとめ

- **鉄は「量」より「吸収率」が重要**：非ヘム鉄の吸収率はわずか2〜5%だが、ビタミンC50mg以上の同時摂取で最大6倍に向上する。紫キャベツの酢漬け1人前（ビタミンC 55mg）を鉄分豊富な主菜と合わせるだけで吸収効率が劇的に改善する
- **タンニンの影響は「食事中の同時摂取」を避ければ大幅に軽減される**：お茶を完全にやめる必要はなく、食事の前後30分〜1時間空ければ十分。ビタミンCの同時摂取でタンニンの阻害効果は相殺可能
- **「鉄鍋＋酸性食材」は隠れた鉄補給源**：酢やトマトを使った料理を鉄鍋で調理すれば、1食あたり+1〜3mgの鉄を上乗せでき、日常の調理器具選びも貧血予防の立派な戦略になる
