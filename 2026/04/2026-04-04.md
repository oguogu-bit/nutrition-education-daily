

# 食物繊維 — 栄養教育ノート 2026-04-04

## 今日の栄養素：食物繊維（Dietary Fiber）

### 働きと効果

食物繊維は「ヒトの消化酵素では分解されない食物成分」の総称であり、**水溶性食物繊維**と**不溶性食物繊維**の2種類に大別されます。水溶性食物繊維（ペクチン、β-グルカン、イヌリンなど）は水に溶けてゲル状となり、糖質や脂質の吸収を緩やかにすることで食後血糖値の急上昇を抑制し、血中コレステロール値の低下にも寄与します。一方、不溶性食物繊維（セルロース、ヘミセルロース、リグニンなど）は水分を吸収して便のかさを増し、腸の蠕動運動を促進して便秘を予防・改善します。さらに、大腸に届いた食物繊維は腸内細菌のエサとなり、**短鎖脂肪酸（酪酸・酢酸・プロピオン酸）** を産生させます。これらは大腸粘膜のエネルギー源となり、腸管バリア機能の強化や全身性の炎症抑制に関与します。欠乏すると便秘・腸内環境の悪化・大腸がんリスクの上昇・2型糖尿病や心疾患のリスク増大が報告されています。食物繊維は過剰摂取による明確な健康障害が少ないため、耐容上限量は設定されていませんが、極端な大量摂取はミネラルの吸収阻害や腹部膨満感の原因となることがあります。

| 対象 | 目標量（以上） | 耐容上限量 |
|------|-------------|-----------|

### 調理・保存による栄養素の変化

食物繊維はビタミンと異なり**加熱による分解がほぼ起こらない**のが大きな特徴です。ただし以下の点に注意が必要です。

- **茹でる**：不溶性食物繊維はほぼ変化なし。しかし**水溶性食物繊維は茹で汁に溶出**します（損失率 約15〜25%）。野菜スープや味噌汁のように汁ごと摂取すれば損失を防げます
- **蒸す・電子レンジ加熱**：水との接触が少ないため、水溶性食物繊維の流出が最小限（損失率 5%以下）。**最も推奨される加熱法**です
- **炒める・焼く**：食物繊維の損失はほぼなし。油と調理することで脂溶性ビタミンの吸収も同時に高められます
- **切り方**：食物繊維そのものは切り方で失われませんが、**大きめに切る**ことで咀嚼回数が増え、満腹中枢が刺激されやすくなります
- **保存**：冷凍保存しても食物繊維量はほぼ変化しません。野菜の冷凍ストックは食物繊維摂取に有効な戦略です

### 吸収効率を上げる食べ合わせ

食物繊維は「吸収される」栄養素ではなく、「大腸で腸内細菌に利用される」栄養素です。そのため**腸内環境を整える組み合わせ**がカギになります。

1. **食物繊維 ＋ 発酵食品（ヨーグルト・味噌・キムチ）**
   → プレバイオティクス（食物繊維）とプロバイオティクス（善玉菌）の**シンバイオティクス効果**。善玉菌に直接エサを届けることで短鎖脂肪酸の産生量が最大化されます

2. **水溶性食物繊維 ＋ 糖質を含む食事**
   → 水溶性食物繊維が胃内でゲル化し、糖質の消化吸収速度を低下させます。**食事の最初に野菜・海藻を食べる「ベジファースト」** の科学的根拠がここにあります

3. **食物繊維 ＋ 十分な水分**
   → 不溶性食物繊維は水分を吸って膨張することで効果を発揮します。水分不足の状態で不溶性食物繊維だけ増やすと、逆に便秘が悪化することがあります。**1日1.5〜2Lの水分摂取**を心がけましょう

### 吸収を妨げる食べ合わせ（注意）

1. **過剰な食物繊維 ＋ ミネラル（鉄・亜鉛・カルシウム）**
   → 食物繊維（特にフィチン酸を含む全粒穀物や豆類）が鉄・亜鉛・カルシウムなどのミネラルをキレート結合し、吸収率を低下させます。鉄剤やカルシウムサプリメントを服用する場合は、高繊維食と**2時間程度ずらして**摂取するのが理想的です

2. **不溶性食物繊維の大量摂取 ＋ 水分不足**
   → 前述のとおり、便が硬くなり便秘を悪化させるリスクがあります。「食物繊維を増やしたのに便秘が治らない」という方の多くがこのパターンです

3. **食物繊維サプリメントの食事中の過剰摂取 ＋ 脂溶性ビタミン（A・D・E・K）**
   → 大量の食物繊維が脂質の吸収を抑制し、結果として脂溶性ビタミンの吸収効率も下がることがあります。通常の食事量であれば問題ありませんが、サプリメントでの大量摂取時は注意が必要です

## 今日のレシピ：鶏もも照り焼き

### 栄養情報（1人前）

| 栄養素 | 量 |
|--------|---|
| カロリー | 320 kcal |
| タンパク質 | 28 g |
| 脂質 | 18 g |
| 炭水化物 | 10 g |
| 食物繊維 | 0 g |
| 鉄分 | 0.7 mg |
| カルシウム | 12 mg |
| ビタミンC | 0 mg |
| ビタミンD | 0.3 μg |
| ビタミンB1 | 0.15 mg |
| オメガ3 | 0.1 g |

### 材料（1人前）

- 鶏もも肉 150g
- 醤油 大さじ1.5
- みりん 大さじ1.5
- 砂糖 小さじ1
- 酒 大さじ1

### 作り方

1. 鶏もも肉の皮をフォークで刺す
2. フライパンに皮目を下にして焼く
3. 裏返してフタをし蒸し焼き
4. 調味料を加えて絡める

### このレシピの栄養学的ポイント

鶏もも肉はアミノ酸スコア100の良質な動物性タンパク質源で、消化吸収率も約97%と非常に優れています。しかし今日のテーマである**食物繊維は0g**。これは動物性食品全般に共通する特徴です。高タンパク・低繊維の食事が続くと腸内の悪玉菌が優勢になりやすいため、**副菜で食物繊維を意識的に補う**ことが、この主菜を最大限に活かす栄養戦略となります。

### 栄養保持・吸収効率UPのためのアレンジ提案

1. **ごぼう・れんこんの付け合わせ追加**：照り焼きのタレで一緒に炒め煮にすることで、不溶性食物繊維を約4〜5g上乗せできます。根菜類の食物繊維は加熱しても損失がほぼなく、タレとの味の相性も抜群。咀嚼回数も増えて満腹感がアップします

2. **千切りキャベツ＋海藻サラダを添える**：キャベツ（不溶性食物繊維）とわかめ（水溶性食物繊維のアルギン酸）を組み合わせることで、**水溶性・不溶性の両方をバランスよく**摂取できます。照り焼きを食べる前にサラダから食べ始めれば「ベジファースト」効果で食後血糖値の上昇も穏やかになります

3. **もち麦ごはんとセットにする**：白米を**もち麦3割混合**に変えるだけで、茶碗1杯あたり約2.3gの水溶性食物繊維（β-グルカン）がプラスされます。照り焼きの甘辛いタレともち麦のプチプチ食感は好相性で、無理なく食物繊維摂取量を底上げできる最も手軽なアレンジです

## 今日のまとめ

- 🥬 **食物繊維は「水溶性」と「不溶性」の両方をバランスよく摂ることが大切**。水溶性は血糖・コレステロール対策に、不溶性は便通改善に効果的
- 🍲 **茹でると水溶性食物繊維が流出する**ため、スープや味噌汁で汁ごと摂取するか、蒸し・レンジ加熱を活用する
- 🍗 **動物性食品（肉・魚）には食物繊維が含まれない**ため、主菜が肉料理の日は副菜・主食で意識的に食物繊維を補うことが重要
