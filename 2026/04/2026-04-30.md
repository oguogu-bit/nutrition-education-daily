

# 総合レビュー — 栄養教育ノート 2026-04-30

## 今日の栄養素：バランスの良い食事の設計と日本人の栄養課題

### 働きと効果

「バランスの良い食事」とは、エネルギー産生栄養素バランス（PFCバランス）を適正範囲に保ちつつ、ビタミン・ミネラル・食物繊維を過不足なく摂取する食事を指す。日本食事摂取基準2020では、エネルギー比率としてタンパク質13〜20%、脂質20〜30%、炭水化物50〜65%を推奨している。いずれかの栄養素が慢性的に不足すると、免疫低下・骨粗鬆症・貧血・疲労感・認知機能低下など多岐にわたる不調を招く。一方、ナトリウム（食塩）やエネルギーの過剰摂取は高血圧・肥満・糖尿病のリスクを高める。現代の日本人に特に顕著な課題は、①カルシウム・鉄・食物繊維の慢性的不足、②食塩の過剰摂取、③若年女性の痩せ（低BMI）とエネルギー不足、④高齢者のタンパク質不足によるフレイルリスクの4点である。栄養素は単独ではなく相互に作用し合うため、「何をどれだけ、何と組み合わせて食べるか」を総合的に設計する視点が不可欠である。

### 調理・保存による栄養素の変化 — 総合まとめ

これまで学んだ29テーマの知見を統合すると、調理による栄養損失は大きく3パターンに分類できる。

1. **水溶性栄養素の溶出**（ビタミンB群・C・カリウム）：茹でると30〜50%が流出。対策は「蒸す」「電子レンジ加熱」「スープごと食べる」。
2. **熱による分解**（ビタミンC・B1）：100℃以上の長時間加熱で20〜40%損失。短時間・低温調理が有効。
3. **酸化による劣化**（ビタミンA・E・ポリフェノール）：空気に触れた状態での長期保存で減少。密封冷凍保存が最適。

逆に、加熱で吸収率が**上がる**栄養素もある。βカロテン（油脂と加熱で吸収率7倍）、リコピン（加熱で細胞壁が壊れ吸収率3〜4倍）がその代表例。調理法は「失わせる」だけでなく「活かす」手段でもある。

### 吸収効率を上げる食べ合わせ — 総合ベスト

1. **鉄 + ビタミンC**：非ヘム鉄の吸収率が3〜6倍向上。ほうれん草のレモン和え、小松菜＋パプリカ炒めなどが実践的。
2. **カルシウム + ビタミンD + ビタミンK2**：腸管でのCa吸収（D）→ 骨への沈着（K2）の黄金トリオ。鮭＋納豆＋小松菜の組み合わせで実現。
3. **タンパク質 + ビタミンB6**：アミノ酸代謝に必須。鶏肉＋バナナ、鮭＋にんにくなど。筋肉合成効率が向上する。

### 吸収を妨げる食べ合わせ（注意）— 総合まとめ

1. **タンニン（緑茶・コーヒー）+ 鉄分**：タンニンが非ヘム鉄とキレート結合し、吸収を最大60%阻害。鉄分食品の食事中〜食後30分は控える。
2. **フィチン酸（玄米・大豆）+ 亜鉛・カルシウム**：ミネラルと不溶性の塩を形成。浸水処理や発酵（味噌・納豆化）でフィチン酸を分解すると改善。
3. **シュウ酸（ほうれん草・たけのこ）+ カルシウム**：不溶性のシュウ酸カルシウムとなり吸収不可に。下茹でで80%のシュウ酸を除去可能。

## 今日のレシピ：鶏むね肉のポン酢蒸し

### 栄養情報（1人前）

| 栄養素 | 量 |
|--------|---|
| カロリー | 210 kcal |
| タンパク質 | 35 g |
| 脂質 | 5 g |
| 炭水化物 | 5 g |
| 食物繊維 | 0 g |
| 鉄分 | 0.5 mg |
| カルシウム | 12 mg |
| ビタミンC | 3 mg |
| ビタミンD | 0.2 μg |
| ビタミンB1 | 0.12 mg |
| オメガ3 | 0.05 g |

### 材料（1人前）
- 鶏むね肉 200g
- ポン酢 大さじ2
- 生姜 1片
- 長ねぎ 適量
- ごま油 小さじ1

### 作り方
1. 鶏むね肉を薄くそぎ切りにする
2. 生姜を千切りにし、鶏肉に乗せてポン酢をかける
3. 耐熱皿に並べラップをして電子レンジで5分加熱
4. 長ねぎをのせてごま油を回しかける

### このレシピの栄養学的ポイント

本レシピは「高タンパク・低脂質」の理想的な主菜であり、PFCバランスではタンパク質比率が約67%と突出して高い。電子レンジ蒸しは茹でる場合に比べてビタミンB群の溶出を最小限に抑えられる優れた調理法である。ただし総合レビューの視点で見ると、**カルシウム・鉄・食物繊維・ビタミンC・ビタミンD**のいずれも大幅に不足しており、日本人に不足しがちな栄養素を副菜で補完する設計が不可欠。absorptionTipにもある通り、ビタミンB1を豊富に含む副菜との組み合わせが疲労回復の鍵となる。

### 栄養保持・吸収効率UPのためのアレンジ提案

1. **小松菜とパプリカの温サラダ添え**：小松菜でカルシウム（170mg/100g）と鉄（2.8mg/100g）を、赤パプリカでビタミンC（170mg/100g）を補完。ビタミンCが鉄の吸収を促進し、本日の主菜単独では不足する3大不足栄養素を一気にカバーできる。ごま油のドレッシングでβカロテンの吸収率も向上。

2. **きのこ（しめじ・まいたけ）の同時蒸し**：鶏肉と一緒にきのこを耐熱皿に並べて蒸すことで、食物繊維（3〜4g追加）とビタミンD（まいたけ：4.9μg/100g）を大幅に補強。追加の手間なく栄養バランスが劇的に改善する。きのこのグアニル酸がポン酢の旨味と相乗効果を発揮し、減塩にも貢献。

3. **雑穀ごはん＋納豆の組み合わせ**：白米を雑穀米に変えることでビタミンB1（糖質代謝に必須）と食物繊維を底上げし、納豆でビタミンK2（骨へのカルシウム沈着促進）・植物性タンパク質・食物繊維をプラス。鶏むね肉の動物性タンパク質と納豆の植物性タンパク質を合わせることで、アミノ酸バランスがさらに最適化される。

## 今日のまとめ

- **日本人の4大栄養課題は「カルシウム・鉄・食物繊維の不足」と「食塩の過剰」**。毎日の食事で副菜の設計を意識するだけで大きく改善できる
- **栄養素は"組み合わせ"で活きる**。鉄＋ビタミンC、カルシウム＋ビタミンD＋K2、タンパク質＋B6の3つの黄金コンビを覚えておこう
- **調理法の選択も栄養戦略の一部**。蒸す・電子レンジ加熱は水溶性ビタミンの損失を最小化する最も実践的な方法。今日のレシピはまさにその好例
