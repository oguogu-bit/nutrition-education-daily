

# 発酵食品・腸内環境 — 栄養教育ノート 2026-04-17

## 今日の栄養素：発酵食品と腸内細菌叢（マイクロバイオーム）

### 働きと効果

発酵食品とは、微生物（乳酸菌・酵母・麹菌・酢酸菌など）の代謝活動によって食品成分が分解・変換されたものを指します。発酵の過程で生じる有機酸・ビタミン類・酵素・短鎖脂肪酸が腸内環境を整え、免疫機能の約70%を担う腸管免疫系（GALT）を賦活します。腸内細菌叢のバランスが崩れるディスバイオーシスでは、便秘・下痢だけでなく、アレルギー疾患・肥満・うつ症状・動脈硬化との関連がメタアナリシスで示されています。一方、発酵食品の過剰摂取では塩分過多（味噌・漬物）やヒスタミン蓄積（チーズ・ワイン）によるアレルギー様症状に注意が必要です。「プロバイオティクス（生きた有用菌を摂る）」と「プレバイオティクス（有用菌のエサとなる食物繊維・オリゴ糖を摂る）」を組み合わせた**シンバイオティクス**が、腸内環境改善の最も効果的な戦略です。

### 調理・保存による栄養素の変化

発酵食品において最も重要なのは**生きた微生物の保持**と**酵素活性の維持**です。

- **加熱の影響**: 乳酸菌・酵母は60℃以上で死滅し始め、70℃以上でほぼ失活します。味噌汁は沸騰直前（約80℃）に火を止めてから味噌を溶くのが鉄則です。ただし、死菌体（バイオジェニクス）にも腸管免疫を刺激する効果があるため、加熱した発酵食品にも一定の価値があります。
- **塩麹の酵素**: 麹菌が産生するプロテアーゼは70℃前後で急速に失活します。肉や魚を塩麹で漬ける際は**冷蔵庫で30分〜一晩**置いて酵素反応を進め、その後に調理するのが最も効率的です。
- **保存方法**: 納豆・味噌・塩麹は冷蔵保存が基本。味噌は冷凍保存しても凍らず（塩分のため）、風味と菌の活性を長期間保てます。ヨーグルトは10℃以下で保存し、開封後は2〜3日以内に食べきるのが理想です。

### 吸収効率を上げる食べ合わせ

1. **発酵食品 ＋ 食物繊維（シンバイオティクス）**: 発酵食品のプロバイオティクス菌と、食物繊維（イヌリン・ペクチン・β-グルカン等）のプレバイオティクスを同時に摂ることで、腸内の有用菌が食物繊維を発酵し短鎖脂肪酸（酪酸・酢酸・プロピオン酸）を産生。酪酸は大腸上皮細胞の主要エネルギー源であり、腸管バリア機能を強化します。**例**: ヨーグルト＋オートミール＋バナナ、味噌汁＋わかめ＋ごぼう。

2. **塩麹 ＋ タンパク質食品**: 麹菌のプロテアーゼがタンパク質をペプチド・アミノ酸に分解し、消化吸収率が向上します。同時にグルタミン酸が生成され旨味もアップ。**例**: 鶏むね肉の塩麹漬け、鮭の塩麹焼き。

3. **納豆 ＋ ビタミンC食品**: 納豆に含まれる非ヘム鉄（100gあたり3.3mg）は、ビタミンCとの同時摂取で吸収率が2〜3倍に向上します。**例**: 納豆＋刻みパプリカ、納豆＋大根おろし。

### 吸収を妨げる食べ合わせ（注意）

1. **発酵食品 ＋ 大量の人工甘味料**: スクラロースやアセスルファムKなどの人工甘味料は、マウス実験で腸内細菌叢のバランスを攪乱する可能性が示唆されています（Nature, 2014）。せっかく発酵食品を摂っても、加糖・人工甘味料入り飲料を大量に飲むと効果が相殺される可能性があります。

2. **抗生物質服用中の発酵食品**: 抗生物質は有用菌・有害菌を問わず殺菌するため、服用中は発酵食品の菌が定着しにくくなります。服用終了後2〜4週間は積極的に発酵食品＋食物繊維を摂り、腸内環境の回復を図りましょう（※服用中でも全く無意味ではなく、抗生物質関連下痢の予防効果はメタアナリシスで支持されています）。

3. **高脂肪食 ＋ 発酵食品**: 高脂肪食は胆汁酸の過剰分泌を招き、腸内の有用菌（特にビフィズス菌）を減少させることが知られています。発酵食品の効果を最大化するには、脂質の過剰摂取を避けることが大切です。

## 今日のレシピ：ブロッコリーの塩麹和え

### 栄養情報（1人前）

| 栄養素 | 量 |
|--------|---|
| カロリー | 55 kcal |
| タンパク質 | 3 g |
| 脂質 | 0.5 g |
| 炭水化物 | 7 g |
| 食物繊維 | 3.5 g |
| 鉄分 | 0.7 mg |
| カルシウム | 35 mg |
| ビタミンC | 85 mg |
| ビタミンD | 0 μg |
| ビタミンB1 | 0.09 mg |
| オメガ3 | 0 g |

### 材料（1人前）

- ブロッコリー 100g
- 塩麹 大さじ1
- オリーブ油 小さじ1
- レモン汁 少々

### 作り方

1. ブロッコリーを小房に分け茹でる（2分）
2. 塩麹・オリーブ油・レモン汁で和える

### このレシピの栄養学的ポイント

ブロッコリーはビタミンC含有量が野菜の中でもトップクラス（100gあたり140mg、茹で後でも約85mg残存）。このビタミンCが非ヘム鉄の吸収を大幅に向上させます。さらに本レシピの主役である**塩麹**は、麹菌由来の酵素とビタミンB群を含む発酵調味料であり、今日のテーマ「発酵食品・腸内環境」の実践そのもの。塩麹の酵素がブロッコリーの細胞壁成分を分解し、栄養素の生体利用率を高めます。加えて、ブロッコリーの食物繊維3.5gがプレバイオティクスとして腸内有用菌のエサとなり、塩麹のプロバイオティクス成分と合わせてシンバイオティクス効果が期待できます。

### 栄養保持・吸収効率UPのためのアレンジ提案

1. **蒸し調理に変更**: 茹でる代わりに電子レンジ（600W・2分）または蒸し器で加熱することで、水溶性ビタミンC・ビタミンB群の流出を最小限に抑えられます。茹でるとビタミンCは約40%損失しますが、蒸し調理なら損失を15〜20%に抑制可能です。蒸し上がりに塩麹を和えれば酵素も生きた状態で摂取できます。

2. **納豆との合わせ小鉢**: 和えたブロッコリーに小粒納豆（1/2パック）を加えると、発酵食品の種類が増え腸内細菌の多様性向上に寄与します。さらに納豆の非ヘム鉄3.3mg/100gにブロッコリーのビタミンCが作用し、鉄の吸収率が2〜3倍に。タンパク質も+4g追加でき、副菜としての栄養バランスが大幅に向上します。

3. **すりごま＋かつお節トッピング**: すりごま（小さじ1）でマグネシウム・カルシウムを補い、かつお節（少々）でイノシン酸の旨味と必須アミノ酸を追加。ごまのリグナン（セサミン）は抗酸化作用があり、ブロッコリーのスルフォラファンとの相乗効果で抗酸化防御力がさらに高まります。

## 今日のまとめ

- **シンバイオティクスを意識する**: 発酵食品（プロバイオティクス）と食物繊維（プレバイオティクス）を同じ食事で組み合わせることが、腸内環境改善の最も確実な方法。今日のレシピは塩麹＋ブロッコリーの食物繊維でまさにシンバイオティクス
- **発酵調味料は「加熱前 or 加熱後」が鍵**: 塩麹や味噌の酵素・有用菌を活かすには、70℃以上の加熱を避けるか、加熱前に漬け置きして酵素反応を十分に進めておく
- **発酵食品は「多種類×毎日」が最強**: 納豆・味噌・塩麹・ヨーグルト・ぬか漬けなど、異なる菌種を含む発酵食品を日替わりで摂ることで、腸内細菌の多様性が高まり免疫機能が強化される
