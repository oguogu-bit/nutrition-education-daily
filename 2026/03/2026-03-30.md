

# 鉄欠乏性貧血の予防 — 栄養教育ノート 2026-03-30

## 今日の栄養素：鉄（Fe）— よくある誤解と吸収効率の最適化

### 働きと効果

鉄は赤血球中のヘモグロビンの構成成分として、全身の細胞に酸素を運搬する極めて重要なミネラルです。体内には約3〜4gの鉄が存在し、その約65%がヘモグロビンに含まれます。鉄が不足すると、酸素運搬能力が低下し、倦怠感・動悸・息切れ・顔面蒼白・集中力低下・爪の変形（スプーンネイル）といった鉄欠乏性貧血の症状が現れます。特に月経のある女性、成長期の子ども、妊婦は需要が増大し不足しやすい栄養素です。一方、過剰摂取（特にサプリメントによる）は鉄沈着症（ヘモクロマトーシス）を引き起こし、肝臓・心臓・膵臓に障害をもたらすリスクがあるため、耐容上限量を守ることが大切です。

### 調理・保存による栄養素の変化

鉄はミネラルであるため、ビタミンのように**加熱で分解されることはありません**。しかし、以下の点に注意が必要です。

- **茹でこぼし**：ほうれん草などを大量の湯で長時間茹でると、溶出した鉄が茹で汁に流出します。短時間の加熱（1〜2分）で引き上げるか、蒸し調理にすると損失を最小限に抑えられます
- **鉄鍋の活用**：鉄製の調理器具を使うと、微量の鉄が食品に溶出し、摂取量を底上げできます。特に酢やトマトなど酸性の食材を煮込むと溶出量が増加します
- **保存による変化**：鉄分自体は冷蔵・冷凍で大きく変化しませんが、**同時に含まれるビタミンC**は時間経過で減少するため、鉄の吸収促進効果を活かすには新鮮なうちに食べるのが理想です

### 吸収効率を上げる食べ合わせ

鉄の吸収効率は食べ合わせで**2〜6倍**変化します。ここが貧血予防の最大のカギです。

1. **非ヘム鉄 ＋ ビタミンC**：ビタミンCは非ヘム鉄（Fe³⁺）を吸収されやすい形（Fe²⁺）に還元します。ほうれん草のおひたしにレモン果汁をかける、小松菜と赤パプリカの炒め物など。吸収率が約3〜6倍に向上するとの研究報告があります
2. **非ヘム鉄 ＋ 動物性タンパク質（MFPファクター）**：肉・魚に含まれるペプチドが非ヘム鉄の吸収を促進します。「ひじきと鶏肉の煮物」「小松菜と豚肉の炒め物」が好例です
3. **鉄 ＋ 有機酸（クエン酸・リンゴ酸）**：酢・柑橘類に含まれる有機酸がキレート作用で鉄を可溶化し、腸管での吸収を助けます。今日のレシピ「紫キャベツの酢漬け」はまさにこの原理を活用しています

### 吸収を妨げる食べ合わせ（注意）

よくある誤解を含め、**鉄の吸収阻害因子**を正しく理解しましょう。

- **タンニン（緑茶・紅茶・コーヒー）**：タンニンが鉄と不溶性の複合体を形成し吸収を阻害します。「食事中にお茶を飲んでも少量なら問題ない」と言われがちですが、紅茶1杯で非ヘム鉄の吸収が**60〜70%低下**するとの報告があります。食事前後30分〜1時間は控えるのが理想です
- **フィチン酸（全粒穀物・豆類・ナッツ）**：フィチン酸は鉄とキレートを形成し吸収を阻害します。ただし「だから玄米は避けるべき」は**誤解**です。発酵（味噌・納豆）、発芽、浸水によりフィチン酸は分解され、ビタミンCの同時摂取で阻害を十分に打ち消せます
- **カルシウム（牛乳・乳製品）**：カルシウムはヘム鉄・非ヘム鉄の両方の吸収を阻害します。鉄分を意識した食事と乳製品は**食事のタイミングをずらす**のが効果的です（例：朝食にヨーグルト、夕食にレバー）

## 今日のレシピ：紫キャベツの酢漬け

### 栄養情報（1人前）

| 栄養素 | 量 |
|--------|---|
| カロリー | 40 kcal |
| タンパク質 | 1 g |
| 脂質 | 0 g |
| 炭水化物 | 8 g |
| 食物繊維 | 1.8 g |
| 鉄分 | 0.3 mg |
| カルシウム | 30 mg |
| ビタミンC | 55 mg |
| ビタミンD | 0 μg |
| ビタミンB1 | 0.04 mg |
| オメガ3 | 0 g |

### 材料（1人前）

- 紫キャベツ 150g
- 酢 大さじ2
- 砂糖 大さじ1
- 塩 小さじ0.5

### 作り方

1. 紫キャベツを千切りにして塩もみ
2. 水気を絞り酢・砂糖で和える
3. 1時間以上冷蔵庫で漬ける

### このレシピの栄養学的ポイント

この副菜は鉄分そのものの含有量は0.3mgと控えめですが、**ビタミンCが55mgと非常に豊富**な点が最大の武器です。鉄を多く含む主菜（レバー・赤身肉・あさりなど）と組み合わせることで、非ヘム鉄の吸収率を飛躍的に高める「吸収サポーター」として機能します。さらに酢の有機酸がキレート作用で鉄の可溶化を助け、酢漬けのため加熱不要でビタミンCの損失が最小限に抑えられます。5日間保存可能な常備菜として、毎日の食事に手軽に添えられる点も実用性が高いです。

### 栄養保持・吸収効率UPのためのアレンジ提案

1. **レモン果汁プラス漬け**：酢の一部をレモン果汁（大さじ1）に置き換えることで、ビタミンCをさらに上乗せし、クエン酸による鉄キレート効果も強化されます。風味もさわやかになり一石二鳥です

2. **かつお節トッピング**：食べる直前にかつお節をひとつまみかけることで、MFPファクター（動物性タンパク質由来の鉄吸収促進ペプチド）を追加できます。かつお節自体にもヘム鉄が含まれており、うま味成分で減塩効果も期待できます

3. **枝豆・蒸し大豆ミックス**：蒸し大豆や枝豆を加えることで、非ヘム鉄（大豆100gあたり約2.0mg）とタンパク質を大幅に強化できます。大豆のフィチン酸は蒸し加工で一部分解されており、紫キャベツのビタミンCが残りの吸収阻害を打ち消してくれるため、理にかなった組み合わせです

## 今日のまとめ

- **鉄の吸収率は食べ合わせで2〜6倍変わる**。鉄分の「量」だけでなく「吸収効率」を意識することが貧血予防の最重要ポイント
- **食事中の緑茶・紅茶・コーヒーは鉄吸収を大幅に阻害する**。鉄を意識した食事の前後30分〜1時間はタンニン飲料を避け、食間に楽しむ習慣をつける
- **ビタミンC豊富な常備菜を冷蔵庫にストックする**。紫キャベツの酢漬けのような副菜を主菜に添えるだけで、毎日の鉄吸収効率を底上げできる
