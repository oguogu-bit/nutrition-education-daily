

# タンパク質 — 栄養教育ノート 2025-05-01

## 今日の栄養素：タンパク質

### 働きと効果

タンパク質は20種類のアミノ酸が鎖状に結合した高分子化合物であり、体重の約16%を占める重要な栄養素です。筋肉・臓器・皮膚・髪・爪・酵素・ホルモン・免疫抗体など、体のあらゆる構造と機能の基盤を担います。体内では常に「合成と分解（ターンオーバー）」が繰り返されており、成人でも1日あたり約180〜200gのタンパク質が分解・再合成されています。食事から摂取されたタンパク質は胃のペプシン、膵臓のトリプシン・キモトリプシン、小腸のペプチダーゼにより段階的にアミノ酸やジ・トリペプチドに分解され、小腸上皮細胞から吸収されます。**欠乏すると**筋肉量の減少（サルコペニア）、免疫力低下、浮腫（アルブミン低下）、創傷治癒の遅延、貧血などが起こります。一方、**過剰な長期摂取**は腎臓への負荷増大（尿素窒素の排泄増加）、カルシウムの尿中排泄促進による骨密度低下リスクが指摘されています。

### 調理・保存による栄養素の変化

タンパク質は他のビタミン類と比べ、加熱による「栄養価の消失」は比較的少ない栄養素です。ただし、以下の点に注意が必要です。

- **加熱（焼く・炒める）**：タンパク質は60〜70℃で変性（立体構造が崩れる）しますが、これはむしろ**消化吸収率を向上**させます。生卵のタンパク質吸収率は約51%ですが、加熱卵は約91%まで上昇します。
- **高温長時間加熱（200℃以上）**：アミノ酸の一部（特にリジン）が糖質と反応する**メイラード反応**が起き、利用効率が低下します。焦げ目は風味になりますが、過度な焦げは避けましょう。
- **煮る・茹でる**：肉の場合、煮汁に水溶性のアミノ酸やペプチドが溶出します（損失率5〜15%）。煮汁ごと摂取するスープや鍋料理が効率的です。
- **保存**：冷凍保存ではタンパク質の栄養価はほぼ変化しません。ただし、解凍時のドリップ（肉汁）にはアミノ酸が含まれるため、ゆっくり冷蔵解凍してドリップを最小限に抑えましょう。

### 吸収効率を上げる食べ合わせ

1. **タンパク質 ＋ ビタミンB6**：ビタミンB6はアミノ酸代謝の補酵素として不可欠です。鶏肉＋バナナ、魚＋にんにくなどの組み合わせが効果的。B6がないとタンパク質を摂っても体内で効率的に利用されません。
2. **動物性タンパク質 ＋ 植物性タンパク質**：米（制限アミノ酸＝リジン）と大豆（リジン豊富だがメチオニンが少ない）を組み合わせると、互いの不足を補完しアミノ酸スコアが実質的に向上します。「ごはん＋味噌汁＋納豆」は日本の伝統的な最適解です。
3. **タンパク質 ＋ ビタミンC**：コラーゲン合成にはビタミンCが必須です。肉料理にレモンを添える、食後に果物を摂るなどで、摂取したタンパク質のコラーゲンへの変換効率が高まります。

### 吸収を妨げる食べ合わせ（注意）

1. **過剰な食物繊維との同時大量摂取**：食物繊維は消化管内でタンパク質を吸着し、吸収率を若干低下させることがあります。特にフィチン酸（玄米・全粒穀物に多い）はミネラルだけでなくタンパク質の消化酵素活性も低下させる報告があります。適量なら問題ありませんが、プロテインサプリとふすまを同時に大量摂取するのは非効率です。
2. **タンニン（高濃度の緑茶・コーヒー）＋ タンパク質**：タンニンはタンパク質と結合して不溶性の複合体を形成し、消化酵素の作用を妨げます。食事中に濃い渋茶を大量に飲むのは避け、食後30分程度あけると良いでしょう。
3. **アルコール＋ タンパク質食**：アルコールは肝臓でのタンパク質合成を抑制し、筋タンパク質の合成速度を最大20〜30%低下させるとの研究があります。トレーニング後の飲酒は筋合成を妨げます。

## 今日のレシピ：豚ヒレの生姜焼き

### 栄養情報（1人前）

| 栄養素 | 量 |
|--------|---|
| カロリー | 280 kcal |
| タンパク質 | 30 g |
| 脂質 | 12 g |
| 炭水化物 | 8 g |
| 食物繊維 | 0.5 g |
| 鉄分 | 0.8 mg |
| カルシウム | 8 mg |
| ビタミンC | 2 mg |
| ビタミンD | 0.1 μg |
| ビタミンB1 | 0.85 mg |
| オメガ3 | 0.08 g |

### 材料（1人前）

- 豚ヒレ肉 150g
- 醤油 大さじ1.5
- みりん 大さじ1
- 生姜 1片
- 玉ねぎ 1/4個

### 作り方

1. 豚ヒレ肉を5mm厚に切り、醤油・みりん・すりおろし生姜に10分漬ける
2. 玉ねぎを薄切りにする
3. フライパンで玉ねぎを炒め、肉を加えて両面焼く
4. タレを加えて絡める

### このレシピの栄養学的ポイント

本レシピは1人前で**タンパク質30g**を摂取でき、1食分として非常に優秀です。さらに豚ヒレ肉は全食品中トップクラスの**ビタミンB1含有量（0.85mg）**を誇り、糖質のエネルギー代謝（ピルビン酸→アセチルCoAの変換）に必須の補酵素として働きます。つまり、ごはんと一緒に食べることで糖質を効率よくエネルギーに変換し、疲労物質の蓄積を防ぎます。生姜に含まれるジンゲロールは胃液分泌を促進し、タンパク質の消化を助ける効果もあります。

### 栄養保持・吸収効率UPのためのアレンジ提案

1. **ブロッコリー添え**：付け合わせにブロッコリー（茹で50g）を加えると、ビタミンC約27mgが追加されコラーゲン合成を促進。さらにブロッコリーのスルフォラファンが抗酸化作用を発揮し、タンパク質の酸化変性を防ぎます。

2. **すりごま＆にんにくプラス**：漬けダレにすりごま大さじ1とにんにく1/2片を追加。ごまのメチオニン・にんにくのビタミンB6がアミノ酸代謝を活性化し、摂取した30gのタンパク質の体内利用効率が向上します。にんにくのアリシンはビタミンB1と結合して脂溶性のアリチアミンとなり、B1の吸収率を飛躍的に高めます。

3. **玄米ごはんとのセット化**：白米を玄米（もしくは白米＋もち麦）に変更することで、米に不足するリジンを豚ヒレ肉が補完し、食事全体のアミノ酸スコアが最大化されます。また食物繊維が加わることで血糖値の急上昇を防ぎ、インスリンの適度な分泌が筋タンパク質合成のシグナルとしても機能します。

## 今日のまとめ

- **タンパク質は「量」だけでなく「質（アミノ酸スコア）」と「食べ合わせ」が重要**。動物性と植物性を組み合わせてアミノ酸バランスを整えよう
- **加熱調理はタンパク質の消化吸収率を向上させる**。ただし焦がしすぎはリジンの損失につながるので中火で丁寧に焼くのがベスト
- **豚ヒレ肉は高タンパク・低脂質・高ビタミンB1の三拍子揃った優秀食材**。にんにくと合わせるとB1の吸収率がさらにアップする

## 明日への一言

> タンパク質を体内で活かすには、脂質の助けも欠かせません。明日は「脂質」の世界を覗いてみましょう——悪者扱いされがちな脂質が、実は細胞膜やホルモンの材料として不可欠な存在であることがわかるはずです。
